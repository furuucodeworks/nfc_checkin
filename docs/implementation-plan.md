# チェックインシステム 実装計画

「作りながら覚える」方針で小さく動くものを積み上げる。各STEPで「動くものが確認できたら次へ」。

正本: [Notion](https://app.notion.com/p/38c3d50a4a03816db3f0e04747e399c6)

詳細な要件は [requirements.md](./requirements.md) を参照する。

作業の日次記録は [journal/](./journal/) に `[日付].md` 形式で残す。

---

## 技術構成

| 項目 | 採用技術 |
|------|----------|
| フロントエンド | Next.js |
| バックエンド | Next.js API Routes |
| データベース | Supabase(PostgreSQL) |
| 認証 | Supabase Auth |
| ホスティング | Vercel |

---

## 実装ステップ

### STEP 1: 環境構築と画面表示

**目安**: 1〜2週間

**作業内容**

- Next.js + Supabaseのプロジェクトを作成
- Vercelにデプロイ
- NFCタグのURLを開いたら「Hello」が表示されるだけの状態を作る

**ゴール**: URLを開いたら画面が出ることを確認

**ステータス**: [ ] 未着手 / [ ] 進行中 / [x] 完了

---

### STEP 2: ログイン機能

**目安**: 1〜2週間

**作業内容**

- Supabase Authでログイン画面を作る
- セッション管理・Cookie設定(365日)
- ログイン後にNFCチェックインURLへ自動リダイレクト

**ゴール**: ログインしてセッションが保持されることを確認

**ステータス**: [ ] 未着手 / [ ] 進行中 / [ ] 完了

---

### STEP 3: チェックイン完了画面

**目安**: 1〜2週間

**作業内容**

- ログイン済みならチェックイン完了画面を表示
- Supabaseからアカウント情報・写真を取得して表示
- 名前・パスの種類・有効期限の表示

**ゴール**: かざす→完了画面が出る最小版の完成

**ステータス**: [ ] 未着手 / [ ] 進行中 / [ ] 完了

---

### STEP 4: DBへの記録

**目安**: 1週間

**作業内容**

- Supabaseにチェックイン記録テーブルを作成
- タッチするたびにDBに記録を保存
- checkin_date_jstの部分ユニークインデックスを設定

**ゴール**: タッチするたびにDBに記録が残る

**ステータス**: [ ] 未着手 / [ ] 進行中 / [ ] 完了

---

### STEP 5: 判定ロジックを一つずつ追加

**目安**: 2〜3週間

**作業内容(順番に進める)**

1. 当日チェックイン済みチェック
2. 支払いステータス確認
3. 有効期限チェック
4. 施設一致確認
5. 時間帯確認(jpholiday使用)

**ゴール**: 要件定義書§19のテストケースがすべて通ることを確認

**ステータス**: [ ] 未着手 / [ ] 進行中 / [ ] 完了

---

### STEP 6: メール送信

**目安**: 1週間

**作業内容**

- 成功時にお客様へメール送信
- 失敗時に施設側へメール送信
- メール送信ログをDBに記録
- 施設ごとの送信元アドレスの設定

**ゴール**: 成功・失敗時に正しい宛先にメールが届くことを確認

**ステータス**: [ ] 未着手 / [ ] 進行中 / [ ] 完了

---

### STEP 7: 管理画面

**目安**: 2〜3週間

**作業内容**

- スタッフ・管理者のログイン機能
- application_idまたは名前での検索機能
- 申し込み内容・アカウント情報の表示
- 支払いステータスの更新機能
- RLS・ロール制御の実装

**ゴール**: スタッフが検索・確認・支払いステータス更新できることを確認

**ステータス**: [ ] 未着手 / [ ] 進行中 / [ ] 完了

---

### STEP 8: テスト・リリース

**目安**: 1週間

**作業内容**

- 要件定義書§19のテストケース（18件）をすべて確認
- 不具合があれば修正
- 本番環境へのデプロイ
- NFCタグに本番URLを書き込んで実際に動作確認

**ゴール**: 実際のタグでチェックインが動くことを確認

**ステータス**: [ ] 未着手 / [ ] 進行中 / [ ] 完了
