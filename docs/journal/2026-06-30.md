## 確認・学習したこと

### `.env.local` と `env.example`

| ファイル | 役割 |
|----------|------|
| `env.example` | テンプレート。GitHub に載せて OK |
| `.env.local` | 本物の Supabase 接続キー。Git に載せない |

- `cp env.example .env.local` = テンプレートをコピーして自分用ファイルを作る
- `NEXT_PUBLIC_SUPABASE_URL` / `NEXT_PUBLIC_SUPABASE_ANON_KEY` = Supabase 接続用
- ローカルは `.env.local`、本番は Vercel の Environment Variables に同じ名前で設定
